PropertyTable v2 - Auto-Detection Demo

This demo shows NumberControl automatically detecting scalar values, arrays, and objects to create appropriate interfaces.

Auto-Detecting Controls

Live Data

Test Cases:

  • intensity (scalar): Regular number control
  • position (array [x,y]): Auto-detects 2-component vector
  • color (object {r,g,b}): Auto-detects 3-component vector
  • size (array [x,y,z]): Auto-detects 3-component vector
  • rotation (array [x,y,z,w]): Auto-detects 4-component vector

Interaction:

  • Click text input → Opens popup slider
  • Right-click → Context menu with PropertyTable
  • Vector components → Individual controls for each component
